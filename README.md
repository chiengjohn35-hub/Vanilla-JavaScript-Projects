- If you have multiple projects in subfolders (for example `/project-1/index.html`), open the playground with the file parameter:
  - CodeSandbox: https://codesandbox.io/s/github/chiengjohn35-hub/Vanilla-JavaScript-Projects?file=/project-1/index.html
  - StackBlitz: https://stackblitz.com/github/chiengjohn35-hub/Vanilla-JavaScript-Projects?file=project-1/index.html

- Use relative asset paths (e.g. `./css/style.css`, `./js/app.js`) so the playground serves everything correctly.
- Remove or hide any secret keys before sharing; these playgrounds run publicly.
